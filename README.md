# Personal Portfolio Website

A modern and responsive personal portfolio website built to showcase my skills, projects, resume, and contact information.

## 🚀 Features

- Responsive modern UI
- About Me section
- Skills showcase
- Project gallery
- Resume download
- Contact form
- Social media links
- Smooth animations
- Mobile-friendly design

## 🛠 Technologies Used

- Angular
- TypeScript
- HTML5
- CSS3
- Tailwind CSS
- JavaScript

## 📂 Sections Included

- Home
- About
- Skills
- Projects
- Resume
- Contact
